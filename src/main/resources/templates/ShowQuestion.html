Creating a simple HTML page is easy. Just follow these steps:

  1. Open a text editor like Notepad.
  2. Write your HTML code, e.g., <!DOCTYPE html><html><head><title>My First Page</title></head><body><h1>Hello, World!</h1></body></html>
  3. Save the file with a .html extension.
  4. Open the saved file in a web browser.
